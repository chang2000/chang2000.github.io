# Oct Reading Clips

## Article Recommendations

- [A Research Guide for Student](https://www.bydewey.com/index.html)
  Provide all the necessary tools for you to conduct research and to present your findings. 很老派但是很经典的一份manuel.

- [优秀的阅读内容如何找](https://blogcn.acacess.com/find-great-content)

- [Six Years with a Distraction Free iPhone](https://medium.com/make-time/six-years-with-a-distraction-free-iphone-8cf5eb4f97e3)
  *Make Time* 作者的作品，在Make Time的专栏下还有很多经典的效率文章，比如Burner List.

- [In what ways can we form useful relationships between notes?](https://medium.com/@nickmilo22/in-what-ways-can-we-form-useful-relationships-between-notes-9b9ec46973c6)
  用relation graph的方式展示不同notes之间的关系，如果有深度使用过Roam Research的经历读起来会更有好处。

- [Starter Stroy: How Sidney Smith start a tour company](https://www.starterstory.com/sidney-smith)

- [How to write like a great entrepreneurs](https://venturehacks.com/writing?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website)

  If you want to be a starter. The main site Venture Hacks is also a place that provides some practical advice. 

- [Why I stopped reading HN](https://sealedabstract.com/rants/why-i-stopped-reading-hn/index.html)

  一篇2010年的老文章，Drew在那个时候就知道逃离因为新人变多而质素下降的平台，哪怕是Hacker News.

- [Well, That Aged Badly](https://forge.medium.com/well-that-aged-badly-dc6f3e378cf9)

  如何面对内容消费时的怀旧倾向

- [“A Typewriter That I Can Carry Anywhere”]("A Typewriter That I Can Carry Anywhere")

  Ulysses的广告文，不过很好的展示了一位业余作家的写作生活。

- 剑桥雅思11-Test 2 - Reading

  提醒自己英语还是要学

- [HOW TO WRITE A GREAT SOFTWARE ENGINEER CV](https://workonomics.co.uk/blog/2020/10/7/cv-advice-product-engineer)

- [Why C is Influential](https://www.youtube.com/watch?v=ci1PJexnfNE&ab_channel=Computerphile)

  三年前看的视频最近又被YouTube神奇算法推荐给我，经过三年本科学习再重看这个视频脉络真的清晰很多。
